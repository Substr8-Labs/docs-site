## Quick Examples

### Generate a Proof

```bash
pip install substr8-core substr8-cli langgraph
python your_agent.py  # outputs runproof.json
```

### Verify a Proof

```bash
substr8 proof verify runproof.json
```

### Inspect Details

```bash
substr8 proof inspect runproof.json --events
```
