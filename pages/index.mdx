# Substr8

**Agents with receipts.**

Substr8 generates cryptographically verifiable receipts for AI agent execution.

---

## The Problem

AI agents are black boxes. When something goes wrong, you can't prove:
- What the agent actually did
- Whether it followed policy
- If the logs were tampered with

---

## The Solution

Wrap your agent with Substr8. Every run produces a **RunProof** — a portable, cryptographically signed receipt containing:

- ✓ **Hash-chained trace** — ordered, tamper-evident events
- ✓ **Merkle root** — single commitment to all events  
- ✓ **Ed25519 signature** — proof of origin
- ✓ **Verification** — anyone can check validity

---

## Quick Start

```bash
# Install
pip install substr8-langgraph

# Wrap your LangGraph
from substr8_langgraph import instrument_graph
instrumented = instrument_graph(graph, agent_id="my-agent", project="my-project")

# Run → produces runproof.json
result = instrumented.invoke(input)

# Verify
substr8 proof verify runproof.json
# → ✓ RunProof: VALID
```
