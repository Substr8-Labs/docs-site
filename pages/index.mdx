# Substr8

**Agent Trust Infrastructure**

> Substr8 turns agent execution into a verifiable, governed, delegable, and auditable workflow.

Frameworks build agents. **Substr8 proves what they did.**

```
Define → Hash → Register → Execute → Enforce → Delegate → Attest → Verify
```

---

## Quick Start

```bash
pip install substr8
```

```python
from substr8 import Substr8Client

client = Substr8Client()

# Start tracking
run = client.run_start(agent_id="my-agent")

# Emit events
client.run_event("tool.called", {"tool": "search"})

# End and verify
client.run_end()
client.verify_run(run.run_id)  # {"verified": True}
```

## Framework Adapters

One line to add observability:

```python
# OpenAI
from substr8.adapters.openai import wrap_openai
client = wrap_openai(OpenAI())

# Anthropic
from substr8.adapters.anthropic import wrap_anthropic
client = wrap_anthropic(Anthropic())

# LangGraph
from substr8.adapters.langgraph import create_callback
graph.invoke(state, config={"callbacks": [create_callback()]})

# CrewAI
from substr8.adapters.crewai import wrap_crew
wrap_crew(crew).kickoff()
```

## Why Substr8?

AI agents have a trust problem. When an agent says "I searched the web and wrote the memory" — did it actually?

Most systems rely on logs. **Substr8 produces cryptographic receipts.**

| Logs | Substr8 |
|------|---------|
| Can be modified | Immutable hash chain |
| Trust the system | Verify independently |
| "It happened" | "Here's proof" |

---

## Core Concepts

### RunProof

A portable, cryptographically verifiable artifact for every governed run.

**[Learn more →](/runproof)**

### TowerHQ

Agent identity registry. Define agents, compute identity hashes, verify authenticity.

### ThreadHQ

Execution observability. Track every prompt, response, tool call, and decision.

### Governance

Policy enforcement. Check capabilities before actions, audit everything.

## The 10 SDK Functions

| Function | Purpose |
|----------|---------|
| `agent_register` | Register agent in TowerHQ |
| `agent_hash` | Compute identity hash |
| `agent_verify` | Verify agent identity |
| `run_start` | Start execution run |
| `run_event` | Emit trust event |
| `run_end` | End run |
| `capability_check` | Check policy |
| `memory_write` | Write to memory |
| `memory_search` | Search memory |
| `verify_run` | Verify hash chain |
