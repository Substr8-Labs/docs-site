# Getting Started

Get agent observability in 5 minutes.

## Installation

```bash
pip install substr8
```

## Quick Start

### Option 1: Direct SDK Usage

```python
from substr8 import Substr8Client

client = Substr8Client()

# Start a run
run = client.run_start(agent_id="my-agent")

# Emit events during execution
client.run_event("tool.called", {"tool": "web_search", "query": "..."})
client.run_event("tool.result", {"result": "..."})

# End the run
result = client.run_end()
print(f"Run {run.run_id} completed")
print(f"Root hash: {result['root_hash']}")

# Verify integrity
verification = client.verify_run(run.run_id)
print(f"Verified: {verification['verified']}")
```

### Option 2: Framework Adapters

#### OpenAI

```python
from openai import OpenAI
from substr8.adapters.openai import wrap_openai

# One line to add observability
client = wrap_openai(OpenAI())

# Use normally - events auto-emitted
response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello"}]
)

client.end_run()
```

#### Anthropic

```python
from anthropic import Anthropic
from substr8.adapters.anthropic import wrap_anthropic

client = wrap_anthropic(Anthropic())

response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello"}]
)

client.end_run()
```

#### LangGraph

```python
from langgraph.graph import StateGraph
from substr8.adapters.langgraph import create_callback

callback = create_callback(agent_id="my-langgraph-agent")

result = graph.invoke(
    initial_state,
    config={"callbacks": [callback]}
)

callback.end_run()
```

#### CrewAI

```python
from crewai import Crew
from substr8.adapters.crewai import wrap_crew

crew = Crew(agents=[...], tasks=[...])
wrapped = wrap_crew(crew)

result = wrapped.kickoff()
# Events auto-emitted, run auto-ended
```

## What Gets Tracked

Every run produces a **RunProof** — a cryptographically verifiable artifact containing:

| Event | When |
|-------|------|
| `run.started` | Execution begins |
| `prompt.sent` | Prompt sent to model |
| `model.responded` | Model response received |
| `tool.called` | Tool invocation |
| `tool.result` | Tool result |
| `memory.write` | Memory stored |
| `decision.recorded` | Decision logged |
| `run.completed` | Execution finished |

## Verify a Run

```python
# Local verification
verification = client.verify_run("run-abc123")

if verification["verified"]:
    print(f"✅ Chain valid: {verification['root_hash']}")
else:
    print(f"❌ Chain broken: {verification['error']}")
```

```bash
# CLI verification
substr8 verify run-abc123
```

## Configuration

```python
from substr8 import Substr8Client, Substr8Config

config = Substr8Config(
    base_url="http://localhost:8100",           # Event Ingest API
    agent_registry_url="http://localhost:8099", # TowerHQ
    gam_url="http://localhost:8091",            # Memory
    acc_url="http://localhost:8096",            # Governance
    api_key="sk-...",                           # Optional auth
    agent_id="my-agent",                        # Default agent
)

client = Substr8Client(config)
```
