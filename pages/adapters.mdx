# Framework Adapters

Pre-built integrations for popular AI frameworks. One line to add observability.

## Available Adapters

| Framework | Wrapper | Auto-Tracks |
|-----------|---------|-------------|
| OpenAI | `wrap_openai()` | prompts, responses, tool calls |
| Anthropic | `wrap_anthropic()` | prompts, responses, tool use |
| LangGraph | `Substr8Callback` | full graph lifecycle |
| CrewAI | `wrap_crew()` | crew execution, tasks |

---

## OpenAI

Wraps the OpenAI client to automatically emit events for all chat completions.

### Installation

```bash
pip install substr8 openai
```

### Usage

```python
from openai import OpenAI
from substr8.adapters.openai import wrap_openai

# Wrap the client
client = wrap_openai(OpenAI(), agent_id="my-openai-agent")

# Use normally - events auto-emitted
response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello"}]
)

# End the run when done
client.end_run()
```

### What Gets Tracked

| Event | When |
|-------|------|
| `prompt.sent` | Before API call |
| `model.responded` | After response |
| `tool.called` | For each tool_call in response |

### With Tools

```python
response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "What's the weather?"}],
    tools=[{
        "type": "function",
        "function": {"name": "get_weather", "parameters": {...}}
    }]
)
# tool.called events auto-emitted for each tool use
```

## Anthropic

Wraps the Anthropic client for Claude API calls.

### Installation

```bash
pip install substr8 anthropic
```

### Usage

```python
from anthropic import Anthropic
from substr8.adapters.anthropic import wrap_anthropic

client = wrap_anthropic(Anthropic(), agent_id="my-claude-agent")

response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello"}]
)

client.end_run()
```

### What Gets Tracked

| Event | When |
|-------|------|
| `prompt.sent` | Before API call |
| `model.responded` | After response |
| `tool.called` | For each tool_use block |

### With Tools

```python
response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    messages=[...],
    tools=[{"name": "get_weather", "input_schema": {...}}]
)
# tool.called events for each tool_use in response
```

## LangGraph

Callback handler for LangGraph/LangChain graph execution.

### Installation

```bash
pip install substr8 langgraph
```

### Usage

```python
from langgraph.graph import StateGraph
from substr8.adapters.langgraph import create_callback

# Create callback
callback = create_callback(agent_id="my-langgraph-agent")

# Build your graph
graph = StateGraph(...)
# ... add nodes, edges ...
compiled = graph.compile()

# Execute with callback
result = compiled.invoke(
    initial_state,
    config={"callbacks": [callback]}
)

# End and verify
callback.end_run()
verification = callback.verify_run()
```

### What Gets Tracked

| Event | When |
|-------|------|
| `run.started` | Graph starts |
| `decision.recorded` | Chain/node transitions |
| `prompt.sent` | LLM calls |
| `model.responded` | LLM responses |
| `tool.called` | Tool invocations |
| `tool.result` | Tool results |
| `run.completed` | Graph finishes |

### Auto-End Mode

```python
# Automatically end run when graph completes
callback = create_callback(
    agent_id="my-agent",
    auto_end_run=True  # Run ends automatically
)
```

## CrewAI

Wraps CrewAI crews for multi-agent observability.

### Installation

```bash
pip install substr8 crewai
```

### Usage

```python
from crewai import Crew, Agent, Task
from substr8.adapters.crewai import wrap_crew

# Create your crew
researcher = Agent(role="Researcher", ...)
writer = Agent(role="Writer", ...)

task1 = Task(description="Research the topic", agent=researcher)
task2 = Task(description="Write the article", agent=writer)

crew = Crew(agents=[researcher, writer], tasks=[task1, task2])

# Wrap for observability
wrapped = wrap_crew(crew, agent_id="my-research-crew")

# Execute
result = wrapped.kickoff()

# Verify
verification = wrapped.verify_run()
```

### What Gets Tracked

| Event | When |
|-------|------|
| `run.started` | Crew kickoff |
| `agent.defined` | Each agent registered |
| `decision.recorded` | Task start/end |
| `run.completed` | Crew finishes |

## Custom Adapters

Build your own adapter using the SDK:

```python
from substr8 import Substr8Client

class MyFrameworkAdapter:
    def __init__(self, framework_client, agent_id="my-agent"):
        self.framework = framework_client
        self.substr8 = Substr8Client()
        self.agent_id = agent_id
    
    def execute(self, input):
        # Start run
        run = self.substr8.run_start(
            agent_id=self.agent_id,
            framework="my-framework"
        )
        
        try:
            # Track events during execution
            self.substr8.run_event("prompt.sent", {"input": input})
            
            result = self.framework.run(input)
            
            self.substr8.run_event("model.responded", {"output": result})
            self.substr8.run_end(status="completed")
            
            return result
        except Exception as e:
            self.substr8.run_end(status="failed")
            raise
```

## Shared Client

Use the same Substr8 client across multiple adapters:

```python
from substr8 import Substr8Client
from substr8.adapters.openai import wrap_openai
from substr8.adapters.anthropic import wrap_anthropic

# Shared client
substr8 = Substr8Client()

# Both use same client
openai_client = wrap_openai(OpenAI(), substr8_client=substr8)
anthropic_client = wrap_anthropic(Anthropic(), substr8_client=substr8)
```
